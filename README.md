# 🔢 Handwritten Digit Classification (SVM)

This project implements a multiclass classification model to recognize handwritten digits (0-9) using a Support Vector Machine (SVM). 

## 🎯 Key Objectives
* Preprocess and flatten 8x8 pixel image data.
* Train an SVM classifier to achieve high accuracy.
* Evaluate performance using a Confusion Matrix and Classification Report.

## 📈 Results
* **Accuracy:** 97%
* **Key Insight:** The model performs exceptionally well on digits like `0` and `6`, but occasionally confuses `5` and `9` due to similar pixel distributions.



## 🛠️ Tech Stack
* Python (Scikit-Learn, NumPy, Pandas)
* Matplotlib & Seaborn (Visualization)
